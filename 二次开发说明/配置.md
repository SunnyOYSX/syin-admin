## 动态配置

```js
// 引入方法
import { config } from '@/libs/util';

// 动态写入配置
config('domain', '127.0.0.1');

// 修改配置
config('domain', '127.0.0.2');

// 读取配置
let domain = config('domain');

console.log(domain); // 127.0.0.2

```
> 动态配置使用vuex存储, 并将保存到本地, 在退出登录时删除
